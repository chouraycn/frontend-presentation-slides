Frontend Slides Skill

演讲者模式
Presenter Mode

按 P 打开演讲者视图 — 实时同步备注、预览、计时器

工作原理

BroadcastChannel
双窗口同步

  • 主屏（观众看到的）发送当前幻灯片索引
  • 演讲者视图实时接收 — 零延迟
  • 无需服务器，完全离线可用
  • 同一台电脑，两个标签页之间通信

演讲者视图功能

四个区域，一览全局

  • 当前幻灯片预览 — 实时缩略图，确认当前内容
  • 下一张预览 — 提前看到下一页，准备转场
  • 演讲备注 — 从幻灯片 data-notes 属性读取
  • 计时器 — 秒表模式，点击可重置

使用方法

添加演讲备注

在 <section> 标签上添加 data-notes 属性：

<section class="slide"
  data-notes="这里写演讲者看到的备注
  可以是提醒词、数据来源或过渡语">
  ...幻灯片内容...
</section>

PPT 转换时，extract_pptx.py v2 会自动读取原始备注
